## 🌟 Key Features

### 🖥️ Interactive Linux-style Terminal
Navigate the portfolio using a custom bash-like terminal. Type `help` to see available commands or explore hidden system features.

### 🔐 Terminal-Based CMS (Full-Stack)
Built-in administrative interface accessible via `su admin`.
- **WYSIWYG Inline Editing**: Real-time content updates directly on the page.
- **Drag-and-Drop Architecture**: Powered by SortableJS for reordering works and certificates.
- **Secure Authentication**: Integrated with Supabase Auth and Row Level Security (RLS).

### 🕸️ Neural Tech Constellation
An interactive D3.js force-directed graph representing technical skills. Each node is interactive, allowing users to grab, pull, and explore the "neural network" of expertise.

### 🎭 Deep-Z Cinematic Parallax
Multi-layered glass-morphism panes that travel at varying speeds during scroll, creating a profound 3D depth illusion powered by **GSAP ScrollTrigger**.

### 🕶️ Matrix Mode (Easter Egg)
Execute the **Konami Code** (`↑ ↑ ↓ ↓ ← → ← → B A`) to bypass the system's security and enter the "Matrix," complete with falling code rain and high-contrast digital aesthetics.

## 🏁 Getting Started

### Prerequisites
- A **Supabase** project for the backend logic and real-time CMS.
- A **Hashnode** blog subdomain for dynamic article fetching.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Naveen-Jayaraj/Portfolio_Website.git
   ```
2. Set up your Supabase environment:
   - Configure `SUPABASE_URL` and `SUPABASE_ANON_KEY` in `script.js`.
   - Update the Hashnode host in `script.js`.
3. Open `index.html` in your browser (Live Server recommended).
